# Recetario
## Proyecto final del bloque 2

En este proyecto se utilizan los comandos y etiquetas basicas en la creacion de una pagina web. Ademas de la mejora de la estetica con CSS

## Características
- Uso de etiquetas semánticas (html, head, body, header, main, section, footer).
- Organización de contenido con títulos (h1, h2, h3).
- Párrafos informativos.
- Listas ordenadas y desordenadas para ingredientes y pasos.
- Imágenes ilustrativas.
- Enlaces de navegación interna.
- Formulario para enviar recetas.
- CSS enfocado en mejorar la estética del sitio mediante colores rosa/rojo, tipografías modernas, diseño tipo tarjetas y uso de sombras y degradados
